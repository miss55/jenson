* TOC
{:toc}

1. 列出session

    ```shell
      tmux ls
    ```

1. 新建session

    ```shell
      tmux new -s <session-name>
    ```

1. 接入session

    ```shell
      tmux attach -t <session-name>
    ```

1. 切换会话

    ```shell
      tmux switch -t <session-name>
    ```

1. 分离session

    ```shell
      tmux detach
    ```

1. 关闭session
    > (原则上不这么操作，关闭后session就没有了)

    ```shell
      exit
    ```

1. 杀死session

    ```shell
      tmux kill-session -t <session-name>
    ```

1. 列出所有快捷键

    ```shell
      tmux list-keys
    ```

1. 列出所有tmux 命令

    ```shell
      tmux list-commands
    ```

1. 列出当前所有tmux 会话信息

    ```shell
      tmux info
    ```
